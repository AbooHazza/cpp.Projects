# C++ Projects

This repository contains a collection of C++ data structure implementations developed during university coursework.

## Files

### 1. linkedlist.cpp

* Basic implementation of a singly linked list
* Includes operations like insertion, deletion, and traversal

### 2. linkedlist2.cpp

* Alternative or extended version of linked list implementation
* May include additional features or different logic

### 3. StackWithLinkedList.cpp

* Stack implementation using linked list
* Supports push and pop operations (LIFO)

### 4. QueueWithLinkedList.cpp

* Queue implementation using linked list
* Supports enqueue and dequeue operations (FIFO)

### 5. Stack_Queue_Using_LinkedList.cpp

* Combined implementation demonstrating both stack and queue using linked list

## How to Run

Compile any file using:
g++ filename.cpp -o output

Then run:
./output

## Purpose

These projects were created to practice fundamental data structures such as linked lists, stacks, and queues using C++.
